# Getting Started

RyunixJS is a lightweight JavaScript framework for building user interfaces. Like React, but smaller and with integrated routing.

> [!WARNING]
> **Integrated Routing** (the legacy way of defining routes manually) is no longer supported. Please use the **App Router** instead.

## Quick Start

<CraQuickStart channel="canary" />

Your app is now running at `http://localhost:3000`

## Your First Component

```jsx
// app/index.ryx
import { useStore } from "@unsetsoft/ryunixjs";

function App() {
  const [count, setCount] = useStore(0);

  return (
    <div>
      <h1>Count: {count}</h1>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}

export default App;
```

## Key Concepts

- **Components** - Functions that return JSX
- **Hooks** - `useStore`, `useEffect`, `useMemo`, etc.
- **Routing** - Built-in App Router, no external packages needed
- **JSX** - Same syntax as React

## Project Structure

```
my-app/
├── app/               # App Router (or src/app/ when rootDir is "src")
│   ├── layout.ryx     # Root layout
│   ├── index.ryx      # Home (/)
│   └── error.ryx      # Global 404 (export NotFound)
├── assets/            # Static files (CRA default)
├── styles/            # Global CSS (optional)
└── ryunix.config.js
```

> [!TIP]
> If `app/` is not at the project root, set `rootDir: "src"` in `ryunix.config.js` and place routes in `src/app/`.
