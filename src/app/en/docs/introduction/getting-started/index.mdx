# Getting Started

RyunixJS is a lightweight JavaScript framework for building user interfaces. Like React, but smaller and with integrated routing.

> [!WARNING]
> **Integrated Routing** (the legacy way of defining routes manually) is no longer supported. Please use the **App Router** instead.

## Your First Component

```jsx
// app/index.ryx
function App() {
  const [count, setCount] = useStore(0);

  return (
    <div>
      <h1>Count: {count}</h1>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}

export default App;
```

## Key Concepts

- **Components** - Functions that return JSX
- **Hooks** - `useStore`, `useEffect`, `useMemo`, etc.
- **Routing** - Built-in App Router, no external packages needed
- **JSX** - Same syntax as React

## Project Structure

```
my-app/
├── app/
│   ├── layout.ryx     # Root layout
│   └── index.ryx      # Root component (/)
├── public/            # Static assets
└── ryunix.config.js   # Configuration
```
