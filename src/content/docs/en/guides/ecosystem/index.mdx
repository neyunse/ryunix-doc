## Hydration boundaries (core)

Two helpers ship in `@unsetsoft/ryunixjs` for SSR + client reconciliation:

| Component | Role |
| :--- | :--- |
| `<ServerBoundary id="‚Ä¶">` | Preserves server HTML inside the subtree during hydration (skips client reconciliation of inner content). See [Server Actions](/en/docs/core/server-actions). |
| `<HydrationBoundary id="‚Ä¶">` | Scopes **local** recovery when markup and client VDOM disagree, without remounting the whole app. |

Tune global behavior in `ryunix.config.js`:

```javascript
{
  hydration: {
    recover: "boundary", // "boundary" | "root" | "none"
    boundaries: "route", // "route" | "server-only" | "all-layouts"
    strict: false,       // verbose mismatch warnings in development
  },
}
```

This docs site wraps each locale‚Äôs docs tree in `<HydrationBoundary id="docs-en">` (see `src/features/docs/LocaleDocLayout.ryx`). Full detail: [Rendering & SSR](/en/docs/core/rendering) and [Configuration](/en/docs/api/configuration).

## Theme system (core)

Ryunix includes a small **light / dark / system** theme API (no third-party dependency):

| Export | Use |
| :--- | :--- |
| `ThemeToggle`, `ThemeInitScript` | Ready-made UI and inline boot script |
| `applyTheme`, `resolveThemeFromCookie`, `setThemeCookie` | Cookie + `class="dark"` on `<html>` |
| `createThemeController` | Shared controller for custom UI |
| `@unsetsoft/ryunixjs/theme-toggle.css` | Styles for `ThemeToggle` |

Semantic page styles from `style: true` still use `class="dark"` on `<html>` for dark mode (see [Configuration](/en/docs/api/configuration)). Built-in layout primitives: `Header`, `Footer`, `Main` ‚Äî documented under [Components](/en/docs/api/components).

## Semantic `style` and typography

`style: true` (or a `style` object) injects `@unsetsoft/ryunixjs/style.css` plus optional typography from the preset:

```javascript
{
  style: {
    padding: "1.5rem",
    maxWidth: "72rem",
    font: "inter", // presets: system | inter | source-sans | nunito
    // or custom / Google Fonts:
    // font: { family: "Brand", google: "Inter:wght@400;600" },
  },
}
```

Presets `inter`, `source-sans`, and `nunito` load from [fonts.bunny.net](https://fonts.bunny.net) (privacy-friendly CDN). Tailwind CRA templates set `style: false` to avoid clashes.
