# Deploying

After development, run a production build and serve the output from `.ryunix/static/` or use the built-in Node server with `ryunix start`. Set `webpack.production: true` in `ryunix.config.js` before shipping.

## Building for Production

### 1. Update Configuration

```javascript
// ryunix.config.js
module.exports = {
  webpack: {
    production: true, // Enable production mode
  },
};
```

### 2. Build

<CodeTabs tabs={[
  { name: "npm", code: "npm run build" },
  { name: "yarn", code: "yarn build" },
  { name: "bun", code: "bun run build" },
  { name: "pnpm", code: "pnpm build" }
]} />

Your production build will be in `.ryunix/static/`.
