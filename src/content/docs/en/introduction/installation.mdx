# Installation

To start building with Ryunix you need Node.js 20, 22, or 24. The fastest path is Create Ryunix App: it scaffolds a project with the App Router, dev/build/start scripts, and ready-to-use config. You can also add Ryunix to an existing project by installing @unsetsoft/ryunixjs and @unsetsoft/ryunix-presets.

## Using Create Ryunix App

The fastest way to get started is by scaffolding a new project automatically. Works with npm, yarn, bun, and pnpm:

## Package managers

You can also install RyunixJS manually into an existing project. It works with all major package managers:

<ManualInstallTabs channel="canary" />

For production apps, omit `@canary` or pass `--latest` to Create Ryunix App so dependencies resolve to the stable npm tag.

## Published packages on npm

RyunixJS is split into focused packages. CRA resolves `latest` or `canary` tags from the registry when scaffolding.

<NpmPackagesTable />

| Install | Command |
| :--- | :--- |
| Scaffold (canary) | `npx @unsetsoft/cra@canary my-app --canary` |
| Scaffold (stable) | `npx @unsetsoft/cra my-app` or `npx @unsetsoft/cra my-app --latest` |
| Core + presets only | See package manager tabs above |
