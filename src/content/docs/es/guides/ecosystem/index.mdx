## Límites de hidratación (core)

Dos helpers en `@unsetsoft/ryunixjs` para SSR + reconciliación en cliente:

| Componente | Rol |
| :--- | :--- |
| `<ServerBoundary id="…">` | Conserva el HTML de servidor del subárbol durante la hidratación. Ver [Server Actions](/es/docs/core/server-actions). |
| `<HydrationBoundary id="…">` | Acota la **recuperación local** cuando el markup y el VDOM del cliente no coinciden, sin remontar toda la app. |

Ajuste global en `ryunix.config.js`:

```javascript
{
  hydration: {
    recover: "boundary", // "boundary" | "root" | "none"
    boundaries: "route", // "route" | "server-only" | "all-layouts"
    strict: false,       // avisos verbosos de desajuste en desarrollo
  },
}
```

Este sitio envuelve cada árbol de docs en `<HydrationBoundary id="docs-es">` (ver `src/features/docs/LocaleDocLayout.ryx`). Más detalle: [Renderizado y SSR](/es/docs/core/rendering) y [Configuración](/es/docs/api/configuration).

## Sistema de tema (core)

Ryunix incluye una API pequeña de tema **claro / oscuro / sistema** (sin dependencias de terceros):

| Export | Uso |
| :--- | :--- |
| `ThemeToggle`, `ThemeInitScript` | UI lista y script inline de arranque |
| `applyTheme`, `resolveThemeFromCookie`, `setThemeCookie` | Cookie + `class="dark"` en `<html>` |
| `createThemeController` | Controlador compartido para UI propia |
| `@unsetsoft/ryunixjs/theme-toggle.css` | Estilos de `ThemeToggle` |

Los estilos semánticos de `style: true` siguen usando `class="dark"` en `<html>` para modo oscuro (ver [Configuración](/es/docs/api/configuration)). Primitivas de layout: `Header`, `Footer`, `Main` — en [Componentes](/es/docs/api/components).

## `style` semántico y tipografía

`style: true` (u objeto `style`) inyecta `@unsetsoft/ryunixjs/style.css` y tipografía opcional del preset:

```javascript
{
  style: {
    padding: "1.5rem",
    maxWidth: "72rem",
    font: "inter", // presets: system | inter | source-sans | nunito
    // o personalizado / Google Fonts:
    // font: { family: "Brand", google: "Inter:wght@400;600" },
  },
}
```

Los presets `inter`, `source-sans` y `nunito` cargan desde [fonts.bunny.net](https://fonts.bunny.net). Las plantillas CRA con Tailwind usan `style: false` para evitar choques.
