## Archivo de configuraciГіn

```javascript

  rootDir: "src",
  ssr: true,
  mdx: false,
  buildDir: ".ryunix",
  favicon: true,
  webpack: {
    production: false,
    resolve: {
      alias: {
        "@": "./src",
        "@/components": "./src/components",
      },
    },
  },
};

```

## Variables de entorno

Crea un archivo `.env`:

```env
RYUNIX_APP_API_URL=https://api.example.com
RYUNIX_APP_KEY=abc123
```

Acceso en cГіdigo:

```jsx
const apiUrl = ryunix.env.RYUNIX_APP_API_URL;
const key = ryunix.env.RYUNIX_APP_KEY;
```

Desde `ryunix.config.js`:

```javascript
const RyunixSettings = {
  env: {
    RYUNIX_APP_KEY: "XXXXXXXXXX",
  },
};
```

```jsx
const key = ryunix.config.env.RYUNIX_APP_KEY;
```

Todas las variables de entorno **deben** empezar por `RYUNIX_APP_` para exponerse al cliente.
