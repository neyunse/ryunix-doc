# Despliegue

Tras el desarrollo, ejecuta un build de producción y sirve la salida desde `.ryunix/static/` o usa el servidor Node integrado con `ryunix start`. Activa `webpack.production: true` en `ryunix.config.js` antes de publicar.

## Compilar para producción

### 1. Actualizar la configuración

```javascript
// ryunix.config.js
module.exports = {
  webpack: {
    production: true, // Enable production mode
  },
};
```

### 2. Build

<CodeTabs tabs={[
  { name: "npm", code: "npm run build" },
  { name: "yarn", code: "yarn build" },
  { name: "bun", code: "bun run build" },
  { name: "pnpm", code: "pnpm build" }
]} />

El build de producción quedará en `.ryunix/static/`.
