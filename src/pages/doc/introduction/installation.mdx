# Installation

## Using Create Ryunix App

The fastest way to get started:

```bash
npx @unsetsoft/cra@latest my-app
cd my-app
npm install
npm run dev
```

## Package Managers

Works with npm, yarn, and bun:

```bash
# npm
npm install @unsetsoft/ryunixjs

# yarn
yarn add @unsetsoft/ryunixjs

# bun
bun add @unsetsoft/ryunixjs
```

## Requirements

- Node.js from 20+ to 22+
